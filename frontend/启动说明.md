# 🚀 前端启动说明

## 快速启动

### 方式1：使用批处理文件（推荐）
双击运行：
```
启动前端.bat
```

### 方式2：使用 CMD 命令行
```cmd
cd frontend
npm run dev
```

### ⚠️ 注意
- **不要使用 PowerShell**（会有语法错误）
- 请使用 **CMD** 或 **批处理文件**

---

## 访问地址

启动成功后，访问：
**http://localhost:5173**

---

## 如果遇到问题

### 1. 端口被占用
```cmd
# 查找占用 5173 端口的进程
netstat -ano | findstr :5173

# 结束进程（替换 PID 为实际进程ID）
taskkill /PID <PID> /F
```

### 2. 依赖缺失
```cmd
cd frontend
npm install
npm run dev
```

### 3. 缓存问题
```cmd
cd frontend
rmdir /s /q node_modules
rmdir /s /q .vite
npm install
npm run dev
```
