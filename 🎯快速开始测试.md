## 💬 需要帮助？

### 常见问题

**Q: 前端打不开？**
A: 确保运行了 `cd frontend && npm run dev`

**Q: MetaMask 连接失败？**
A: 检查是否在 Sepolia 网络

**Q: 交易失败？**
A: 检查余额是否充足（需要 ETH 支付 Gas）

**Q: 看不到游戏？**
A: 刷新页面或检查浏览器控制台
